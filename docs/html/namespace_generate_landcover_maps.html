Detailed Description

Import landcover raster maps into a GRASS location and generate landcover, lai, and impervious coverage maps.

Pre conditions

  1. Configuration file must define the following sections and values: 'GRASS', 'GISBASE' 'SCRIPT', 'ETC'
  1. The following metadata entry(ies) must be present in the manifest section of the metadata associated with the project directory: grass_dbase grass_location grass_mapset landcover
  1. The following metadata entry(ies) must be present in the study area section of the metadata associated with the project directory: landcover_type

Post conditions

  1. The following raster datasets will be created in the GRASS location: landcover_raw landcover impervious lai
  1. Will write the following entry(ies) to the manifest section of metadata associated with the project directory: lancover_rule landcover_impervious_rule landcover_lai_rule

Usage:

1 PYTHONPATH=${PYTHONPATH}:../EcohydroWorkflowLib python2.7 ./GenerateLandcoverMaps.py -p ../../../scratchspace/scratch7
Note
EcoHydroWorkflowLib configuration file must be specified by environmental variable 'ECOHYDROWORKFLOW_CFG', or -i option must be specified.
