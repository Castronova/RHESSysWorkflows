Detailed Description

Import percent sand and percent clay raster maps into a GRASS location and generate soil texture map using r.soils.texture.

Pre conditions

  1. Configuration file must define the following sections and values: 'GRASS', 'GISBASE'
  1. The following metadata entry(ies) must be present in the manifest section of the metadata associated with the project directory: grass_dbase grass_location grass_mapset soil_raster_avgsand soil_raster_avgclay
  1. Requires r.soils.texture GRASS extension: http://grasswiki.osgeo.org/wiki/GRASS_AddOns#r.soils.texture

Post conditions

  1. The following raster datasets will be created in the GRASS location: soil_raster_avgclay, soil_raster_avgsand, soil_texture

Usage:

1 PYTHONPATH=${PYTHONPATH}:../EcohydroWorkflowLib python2.7 ./GenerateSoilTextureMap.py -p ../../../scratchspace/scratch7
Note
EcoHydroWorkflowLib configuration file must be specified by environmental variable 'ECOHYDROWORKFLOW_CFG', or -i option must be specified.
